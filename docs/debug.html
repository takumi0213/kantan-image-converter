かんたん画像変換 DEBUG

このページは開発者向けのテスト・デバッグ用です。各画像を右クリックして拡張機能の動作を検証してください。

1. アニメーション画像（変換せず元のまま保存されるべき）

2. data URI 画像（Canvas経由で変換されるべき）

3. SVG 画像（変換せず元のまま保存されるべき）

4. 透過画像（JPG変換時に白背景が付くべき）

5. ファイル名エッジケース

サニタイズ処理の動作確認。以下はdata URIなので日時フォールバック（YYYYMMDD_HHMMSS）が使われます。

6. 手動テストマトリクス

テストケース PNG変換 JPG変換 WebP変換 備考
通常画像（JPG元） 正常に変換されること
通常画像（PNG元） 正常に変換されること
通常画像（WebP元） 正常に変換されること
アニメーションGIF 変換せず元のGIFとして保存
アニメーションWebP 変換せず元のWebPとして保存
SVG画像 変換せず元のSVGとして保存
透過PNG → JPG —— 透過部分が白背景になること
