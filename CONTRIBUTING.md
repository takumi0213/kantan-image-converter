# コントリビューションガイド

かんたん画像変換へのコントリビューションを歓迎します。

## Issue

- **バグ報告**: Issue テンプレートに沿って報告してください。
- **機能提案**: 提案は歓迎しますが、コアコンセプト（画像の変換・保存）に沿わない提案は採用されない場合があります。
- **セキュリティの問題**: Issue には投稿せず、[Private Vulnerability Reporting](https://github.com/takumi0213/kantan-image-converter/security/advisories/new) から報告してください。

## Pull Request

1. Issue で事前に相談することを推奨します（特に機能追加の場合）
1. `main` ブランチから作業用ブランチを作成してください
1. PR テンプレートに沿って説明を記載してください
1. 以下の確認を行ってください
   - 通常のWebページで各フォーマット（PNG/JPG/WebP）の変換・保存が正常に動作する
   - `chrome://` 等の制限ページでエラーが発生しない
   - 既存の動作に影響を与えない

## 開発方針

- **外部依存なし**: npm / CDN 等の外部ライブラリは使用しません
- **シンプルさの維持**: コアコンセプトから逸脱しない範囲での改善を重視します
- **セキュリティ最優先**: URLスキーム検証、ファイル名サニタイズ等のセキュリティ要件は変更できません

## 開発環境のセットアップ

1. リポジトリをフォーク・クローン
1. `chrome://extensions` でデベロッパーモードをON
1. 「パッケージ化されていない拡張機能を読み込む」でクローンしたフォルダを選択
1. コード変更後は拡張機能をリロードして動作確認
