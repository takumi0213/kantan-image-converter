# サポート

## 質問・サポートを受ける前に

まず以下をご確認ください。

- [README.md](README.md) — インストール方法・使い方・対応ブラウザ・よくある制限事項
- [既存の Issues](https://github.com/takumi0213/kantan-image-converter/issues) — 同様の質問や報告が既にないか検索してください

## バグ報告・機能提案

[GitHub Issues](https://github.com/takumi0213/kantan-image-converter/issues/new/choose) からテンプレートを選択して報告・提案してください。

| テンプレート | 用途 |
| --- | --- |
| バグ報告 | 拡張機能が正常に動作しない場合 |
| 機能提案 | 新機能・改善アイデアの提案 |

> [!NOTE]
> 本プロジェクトは個人が業務外で開発・維持しているため、対応に時間がかかる場合があります。あらかじめご了承ください。

## セキュリティ上の問題

セキュリティ上の問題（脆弱性など）は **Issue に公開投稿しないでください**。

[Private Vulnerability Reporting](https://github.com/takumi0213/kantan-image-converter/security/advisories/new) から非公開で報告してください。詳細は [SECURITY.md](SECURITY.md) を参照してください。

## 対応外のサポート

以下については対応していません。

- Firefox・Safari など Chromium ベース以外のブラウザへの対応
- [Chrome Web Store のレビュー](https://chromewebstore.google.com/detail/lecilkeobjibofjaoadlelgfiipicdlo) への返信はしていません。バグ報告・機能提案は GitHub Issues をご利用ください
