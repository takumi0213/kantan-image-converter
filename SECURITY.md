## 脆弱性の報告

セキュリティ上の問題を発見した場合は、**Issue に公開投稿しないでください**。

GitHub の [Private Vulnerability Reporting](https://github.com/takumi0213/kantan-image-converter/security/advisories/new) から非公開で報告してください。

報告にはできる限り以下の情報を含めていただけると助かります。

- 問題の概要
- 再現手順
- 影響範囲の見積もり
- 可能であれば修正案

報告を受け取り次第確認し、対応します。
